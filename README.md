# mcp-intent-firewall

Guarded AI agent assignment scaffold for MCP intent enforcement.

This repo is organized as a Bun workspace:

```txt
apps/api            # Bun API server
apps/web            # Vite + React dashboard
packages/shared     # Shared TypeScript types
packages/custom-mcp # Custom MCP server package
prisma              # Database schema will be added later
```

## Commands

```bash
bun install
bun run dev:api
bun run dev:web
bun run custom:mcp
bun run typecheck
```
